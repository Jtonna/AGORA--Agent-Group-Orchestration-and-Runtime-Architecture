# AGORA -- Agent Group Orchestration and Runtime Architecture

AGORA is a framework for orchestrating groups of AI agents. It provides a shared communication bus (email server), XML-based agent configuration, and a terminal dashboard for monitoring agent activity. Published as the `agora-framework` npm package.

## Quick Start

```bash
npm install agora-framework
npx agora init
npx agora start
npx agora mail
```
